GHCi:
- Wenn GHCi module nicht importieren kann, ist eine Lösung der Start von GHCi
  über Cabal, wobei die entsprechenden module als Parameter angegeben werden
  müssen, z.B. um Data.Regex.Base importieren zu können:
  > cabal repl --build-depends regex-base
